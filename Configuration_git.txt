1.让Git显示颜色，会让命令输出看起来更醒目：
                  git config --global color.ui true                  #高亮显示
				  
				  
2.忽略某些文件时，需要编写.gitignore；
.gitignore文件本身要放到版本库里，并且可以对.gitignore做版本管理！


3.需要敲一行命令，告诉Git，以后st就表示status：
                  git config --global alias.st status                #简化命令
